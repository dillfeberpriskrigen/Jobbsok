
# Jobbsök (projektnamn)

Jobbsök är en databas som hämtas från Arbetsförmedlingens jobbtechdev api. Tanken är att göra det lättare för arbetssökare att hitta och prenumerera på specifika joblistingar vilket i sin tur ska underlätta sökfriktioner. 

# Feature list (att göra lista)

## Keywords

Implementera en NLM som skapar en lista av nyckelord som bör finnas i personligt brev.
Denna bör separera arbetsbeskrivning som användare kan sortera på och personbeskrivning som sökaren sorterar på. Exempelvis genom att identifiera "Arbetsbeskrivning" och "om dig" rubriker.

## Prenumeration

Implementera ett sätt att prenumerera på annonser och bli mailad när ens nyckelord uppdateras.
